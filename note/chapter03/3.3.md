# 3.3 平衡查找树
## 3.3.1 2-3查找树
* 3-结点：含有两个键和三个链接。
* 一棵2-3查找树或为一棵空树，或由2-结点、三结点组成。
* 一棵完美平衡的2-3查找树中的所有空链接到根结点的距离都应该是相同的。
* 查找操作
  * 向2-结点中插入新键
  * 向一棵只含有一个3-结点的树中插入新键
  * 向一个父结点为2-结点的3-结点中插入新键
  * 向一个父结点为3-结点的3-结点中插入新键
  * 分解根结点
* 在一棵大小为N的2-3树中，查找和插入操作访问的结点必然不超过lgN个。
